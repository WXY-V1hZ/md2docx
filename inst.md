我要使用pandoc+lua将md转为docx，我想在转换前先对markdown做预处理，具体包括：

- 标题、图、表自动编号/修复编号
- 图如果有title则将title替换到alt，没有则使用alt，二者都没有则只显示序号

预处理具体实现如下
标题自动编号前，先对所有的标题进行标准化处理，将所有标题中，对最小一级的级数-1记为n，将所有标题减 n-1 级，例如对于：

```markdown
##

###

####
```

会处理为：

```markdown
#

##

###
```

同时对于一个标题下的所有子标题，也会进行同样的处理，例如：

```markdown
##

####

#####
```

会处理为：

```markdown
##

###

####
```

然后进行自动编号，编号规则为：

- 一级标题编号为 1，二级标题编号为 1.1，以此类推
- 编号格式为 1.1.1，即 上一级编号.当前级编号
